GitHub Upload Guide

Simple way to publish this journal website.

  1. Create a new GitHub repository.
  2. Name it anything, for example: trading-journal.
  3. Upload this file as index.html.
  4. Go to repository Settings.
  5. Open Pages.
  6. Under source, choose your main branch.
  7. Save, then GitHub will give you your live website link.
